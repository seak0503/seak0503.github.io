はじめに

ここではgit rebase -i [commit]で、対話式に過去のコミットを 編集方法を説明します。

1つ前のコミットの編集はgit commit --amendでもできるのでここでは 試しにに2つ前のコミットの編集をしてみます

editの説明

editする

ファイルを編集する
